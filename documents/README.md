# Documents Klasörü

Bu klasör OstWindGroup veb saytında istifadəçilərə təqdim edilən yüklənə bilən sənədləri saxlayır.

## Mövcud Sənədlər

### 1. Qəbul Başvuru Formu
- **Fayl:** `application-form.pdf`
- **Təsvir:** Universitetə qəbul üçün lazım olan başvuru forması
- **Format:** PDF

### 2. Nümunə Müraciət Məktubu
- **Fayl:** `sample-letter.docx`
- **Təsvir:** Universitetə müraciət məktubunun nümunəsi
- **Format:** Word (DOCX)

### 3. Viza Rehberi
- **Fayl:** `visa-guide.pdf`
- **Təsvir:** Ukrayna tələbə vizası almaq üçün tam rehber
- **Format:** PDF

### 4. Konaklama Rehberi
- **Fayl:** `accommodation-guide.pdf`
- **Təsvir:** Ukraynada yaşayış yeri tapmaq üçün rehber
- **Format:** PDF

### 5. Burs və Maliyyə Dəstəyi
- **Fayl:** `scholarship-guide.pdf`
- **Təsvir:** Mövcud burs imkanları və maliyyə dəstəyi
- **Format:** PDF

### 6. Diplom Təsdiqi Rehberi
- **Fayl:** `diploma-recognition.pdf`
- **Təsvir:** Diplomların Azərbaycanda təsdiqi prosesi
- **Format:** PDF

## Qeyd

Bu sənədlər nümunə məqsədilə yaradılmışdır. Real sənədlər müvafiq məzmunla əvəz edilməlidir.

## Fayl Formatları

- **PDF:** Oxumaq üçün Adobe Reader və ya digər PDF oxuyucular
- **DOCX:** Microsoft Word və ya Google Docs ilə açıla bilər

## Yeniləmə

Sənədlər müntəzəm olaraq yenilənməlidir və güncel məlumatları əks etdirməlidir.
